1.- Crear la carpeta "components", vite no la crea por defecto
2.- Crear los componentes necesarios, como "ContactForm", estos componentes tienen el html especifico para el componente que se quiere crear
3.- Importar el componente en "App.jsx", ese el archivo principal que se ejecuta al acceder a la pagina
Al crear el formulario de esta forma se crea solo de forma visual, no tiene ninguna funcionalidad real, para eso es necesario usar el hook "useState" de react, integrándolo dentro del componente
